ROBOJOURNAL 0.2 CHANGELOG

 

Release Date: 4/16/2012

#######################################################

The following changes were made in this release:

1.       Added Time support

a.       Added  24-hour clock support

2.       Added Database upgrade tool

a.       Added tool to modify journal databases by adding additional column for time

b.      Added upgrade check when journals are loaded

3.       Added Document theming support

a.       Font and font size

b.      Font color

c.       Background image support on viewer and entry tree panes

d.      Tile background image

4.       Improved date formatting

a.       Added the ability to format date according to USA, European, or ISO-8601 format

b.      Added date format example

5.       Modified Config module

a.       Added Journal and Appearance tabs

b.      Moved Journal-related features to Journal tab

c.       Placed theme-related options on Appearance tab

d.      Removed icons from original General and Database tabs

e.      Added an option to disable alternate row colors

f.        Added the ability to toggle text on toolbar icons

g.       Changed wording on toolbar docking default option label

h.      Moved Entry Range group to Journal tab from General tab

i.         Added placeholder checkbox for rich text format (Disabled, not yet implemented)

6.       Sort by day is no longer default behavior

7.       Fixed parent bug (that caused centering issues) on Login form, Config, About, and Editor modules

8.       Added exit confirmation dialogs for Editor and MainWindow modules

9.       Added print support for journal entries

10.   Changed previous/next shortcut keys to left arrow/ right arrow
